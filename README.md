# ¡Hola! 👋 Soy enriqueav99

**Platform engineer** — homelab, Kubernetes y automatización con IA.

</div>

Llevo un homelab autoalojado (Docker Compose + Traefik + Authentik + CrowdSec + Prometheus/Grafana/Loki) y un clúster k3s gestionado con ArgoCD, y encima construyo herramientas para controlarlo todo — la mayoría con ayuda de [Claude Code](https://claude.ai/code).

## 🛠️ Lo que estoy construyendo

- **[claude-telegram-gateway](https://github.com/enriqueav99/claude-telegram-gateway)** — pasarela de Telegram para Claude Code: habla con Claude desde el móvil, ejecuta comandos, gestiona archivos y recibe alertas de monitorización, todo desde un chat.
- **[telegram-bot](https://github.com/enriqueav99/telegram-bot)** — bot de Telegram para controlar y monitorizar mi homelab, con módulos activables en caliente desde `/panel`.
- **[discord-bot](https://github.com/enriqueav99/discord-bot)** — bot de Discord para servidores pequeños: música, mini-juegos, casino y moderación con logs de auditoría.
